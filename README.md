# Nodejs Hello World
- Create git remote repo
- Clone git repo
- Config env
```sh
cd nodejshw
npm init
```
- Hit enter n times
```sh
description: Nodejs Hello World recipe
author: ralarconb
```
- Install extensions
```sh
npm install express nodemon
```
- Open package.json and edit scripts
  - Remove test
  - Add start
- Start the app
```sh
npm start
```
- http://localhost:3000
